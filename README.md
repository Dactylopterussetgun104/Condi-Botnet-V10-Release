*   **Real-time Traffic Monitoring:** Track data usage on your network endpoints to identify traffic spikes.

*   **Automated Data Collection:** Schedule routine scans to receive updates on your network status.

## 🛠 Using the Botnet Dashboard

The dashboard organizes your network data into clear sections. When you start the application, the system automatically begins its initial scan. You will see a list of detected nodes on the left sidebar. Click any node to view detailed performance metrics. 

The primary graph displays traffic flow over the last hour. Use the slider beneath the graph to adjust the timeframe of the history. If you need to export data, use the Save Logs button at the bottom of the window. This saves your current session metrics as a text file for review later.

## ⚙️ Configuring Network Nodes

1. Open the Settings tab in the application.
2. Enter the IP address of the node you want to monitor.
3. Assign a name to the node for easy identification in your list.
4. Click Add Node.
5. The application will ping the address to confirm connectivity.
6. Once the dot next to the node name turns green, the system is actively monitoring that connection.

## 🛡 Security and Best Practices

This software performs network-level operations. Ensure you run the application on hardware that you control. Restrict access to the configuration files to prevent unauthorized users from editing your list of monitored nodes. Keep your operating system updated to ensure that all network drivers remain compatible with the metrics tools. If the application encounters an error, check the logs by opening the application folder and selecting the logs sub-folder.

## ❓ Frequently Asked Questions

**Does the software require administrator rights?**
Yes. Administrator rights allow the application to access network adapters and gather traffic statistics accurately.

**How do I update the software?**
When a new version releases, visit the download link, download the installer, and run it. The new version will automatically replace the old files while keeping your settings intact.

**Can I run this on multiple machines?**
Yes. You can install the software on as many Windows machines as you need to monitor your network infrastructure.

**What do the colors in the dashboard mean?**
Green indicates a stable connection. Yellow indicates high latency or minor packet loss. Red indicates a lost connection.

**Where does the application store its settings?**
The application saves a local configuration file in the program folder that stores your node list and display preferences.
